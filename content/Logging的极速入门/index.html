Logging的极速入门

logging模块是Python内置的标准模块，主要用于输出运行日志，可以设置输出日志的等级、日志保存路径、日志文件回滚等；相比print，具备如下优点：

  1. 可以通过设置不同的日志等级，在release版本中只输出重要信息，而不必显示大量的调试信息；
  2. print将所有信息都输出到标准输出中，严重影响开发者从标准输出中查看其它数据；logging则可以由开发者决定将信息输出到什么地方，以及怎么输出；

引用python logging模块 - dahu1 - 博客园 (cnblogs.com)

导入

import logging

输出

输出级别（优先度从低到高）为：debug => info => warning => error => critical

Debug

# 打印调试信息
logging.debug(word)

Info

# 打印正常的信息
logging.info(word)

Warning

# 打印警告信息
logging.warning(word)

Error

import traceback

# 打印错误信息
logging.error(word)
# 获取错误信息，在try,except使用最佳
logging.error(traceback.format_exc())

Critical

# 打印致命错误信息
logging.critical(word)

配置

调用basicConfig，传入参数即可，如：

logging.basicConfig(level=logging.DEBUG,format='%(asctime)s %(filename)s[line:%(lineno)d] %(levelname)s %(message)s',datefmt='%a, %d %b %Y %H:%M:%S',filename='logger.log')

参数细节如下：

  • filename：指定输出日志的文件名
  • filemode：和file函数意义相同，指定日志文件的打开模式，’w’或者’a’，不传默认为a。’w’的时候，日志会覆盖书写，’a’是保留前面的日志，并将新的日志存放到文件里面
  • format：指定输出的格式和内容
    • 参数：作用
    • %(levelno)s：打印日志级别的数值
    • %(levelname)s：打印日志级别的名称
    • %(pathname)s：打印当前执行程序的路径，其实就是sys.argv[0]
    • %(filename)s：打印当前执行程序名
    • %(funcName)s：打印日志的当前函数
    • %(lineno)d：打印日志的当前行号
    • %(asctime)s：打印日志的时间
    • %(thread)d：打印线程ID
    • %(threadName)s：打印线程名称
    • %(process)d：打印进程ID
    • %(message)s：打印日志信息
  • datefmt：指定时间格式，同time.strftime()
  • level：设置日志级别，默认为logging.WARNNING
  • stream：指定将日志的输出流，可以指定输出到sys.stderr，sys.stdout或者文件，默认输出到sys.stderr，当stream和filename同时指定时，stream被忽略

Demo

可以将logging封装成一个类，然后调用类的方法即可，详细如下，可直接复制

# -*- encoding=UTF-8 -*-
import logging
import os
import datetime


class LoggingProxy:
    '''
    日志代理类
	
	@author: xxxxxxxx
    @time: xxxxxxxxxx
    '''

    def __init__(self):
        today = datetime.date.today()
        year = today.year
        month = today.month
        # 检测Log文件是否存在，不存在则创建
        path = "./src/log/"+str(year)+"/"+str(month)+"月"+"/"
        filename = str(today)+".log"
        # 文件夹路径不存在则创建
        if(not os.path.exists(path)):
            os.makedirs(path)
        # 文件不存在则创建
        if(not os.path.exists(path+filename)):
            with open(path+filename,'w',encoding="UTF-8") as f:
                f.close()
        logging.basicConfig(level=logging.INFO,
                            format='%(asctime)s %(filename)s[line:%(lineno)d] %(levelname)s %(message)s',
                            datefmt='%a, %d %b %Y %H:%M:%S',
                            filename=path+filename) # logging的基础配置

    def debug(self, word):
        '''
        打印调试信息

        :param word: 调试信息
        '''
        logging.debug(word)
        print(word)

    def info(self, word):
        '''
        打印正常的信息

        :param word: 正常的信息
        '''
        logging.info(word)
        print(word)

    def error(self, word , Exception):
        '''
        打印错误信息

        :param word: 错误信息
        :param Exception: expect中的Exception
        '''
        logging.error(word)
        logging.error(Exception)
        print(word)
        print(Excetion)

    def warning(self, word):
        '''
        打印警告信息

        :param word: 警告信息
        '''
        logging.warning(word)
        print(word)

    def critical(self, word , Exception):
        '''
        打印致命错误信息

        :param word: 致命错误信息
        :param Exception: expect中的Exception
        '''
        logging.critical(word)
        logging.critical(Exception)
        print(word)
        print(Exception)

    def start(self):
        '''
        开始
        '''
        logging.info("======= start =======")

    def end(self):
        '''
        结尾
        '''
        logging.info("======= end =======")
