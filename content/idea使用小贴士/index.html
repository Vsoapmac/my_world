idea使用小贴士

idea是一款非常好用的工具，但是需要许许多多的设置，该记录用于在使用idea的时候，作为指路使用

修改运行的虚拟内存加快运行速度

该方法用于修改idea的运行内存以便加快运行速度，代价是电脑运行变慢

配置文件路径：D:\software\ideaIU-2020.1.3.win\bin\idea.exe.vmoptions和idea64.exe.vmoptions
用记事本打开，修改
-Xms2048m
-Xmx2048m

加快maven搭建速度/安装Maven Helper插件

搜索栏搜索Maven，编辑Maven home directory和User settings file改成自己的
Maven —> Runner —>VM options 添加 -DarchetypeCatalog=local

File --> Settings --> Plugins --> 搜索Helper
安装Maven Helper

文件编码

File --> Settings --> Editor --> File Encodings
全部修改为UTF-8

log文件识别

在输出log文件时，idea无法打开idea查看Log文件，这一步是可以让idea识别.log文件

File --> Settings --> Editor --> File Types

点击Recognized File Types的 “+” ，在name上输入*.log，Description输入日志文件Apply即可

关闭自动更新

Settings --> Appearance & Behavior --> System Settings --> Updates
取消勾选Automatically check updates

使用eclipse快捷键插件

idea自带eclipse的快捷键，但似乎并不好用。Eclipse KeyMap Plugins是最接近eclipse的快捷键的一个非常好用的插件

File --> Settings --> Plugins --> 搜索eclipse --> Eclipse keymap install
File --> Settings --> KeyMap --> eclipse

使用Vue插件(步骤和eclipse差不多，插件名字为Vue.js)

使用热部署插件(JREbel)

在插件里安装Jrebel

在网站上生成GUID

https://www.guidgen.com/

在TeamURL上添加

https://jrebel.qekang.com/{GUID} GUID为刚刚在网站上生成的GUID

邮件名字填写后点击激活，在设置栏里修改为work offline

设置自动导包

可以在编辑类时自动导入包名或者消除包

File --> Settings --> Editor --> General --> Auto Import

勾选 Add unambiguous imports on the fly和Optimize imports on the fly(for current project)

忽略大小写提示

无论大小写都会有相应的提示

File --> Settings --> Editor --> General --> Code Completion

取消Match case的勾选，取消Show the parameter info popup in xxx ms的勾选(加快提示速度)

提示快捷键修改

修改为eclipse常用的alt+/

File --> Settings --> Keymap --> Code --> Code Completion
移除Cyclic Expand Word的快捷键,修改Basic的快捷键为ALT+/

设置syso —> system.out.println()

idea默认是sout,使用eclipse的习惯

File --> Settings 搜索live并找到Live Templates --> output -->sout改为syso
