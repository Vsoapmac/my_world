python翻译

translate

python的google翻译模块，但是有次数限制

安装

pip install translate

引入

from translate import Translator

英译汉

translator2CN = Translator(from_lang='english',to_lang='chinese')
translator2CN.translate("How the financial crisis was caused")
输出：
金融危机是如何引起的

汉译英

translator2EN = Translator(from_lang='chinese',to_lang='english')
translator2EN.translate("金融危机是如何造成的")
输出：
How the financial crisis was caused
