python的邮件发送指南

SMTP是一种提供可靠且有效的电子邮件传输的协议。SMTP是建立在FTP文件传输服务上的一种邮件服务，主要用于系统之间的邮件信息传递，并提供有关来信的通知。SMTP独立于特定的传输子系统，且只需要可靠有序的数据流信道支持，SMTP的重要特性之一是其能跨越网络传输邮件，即“SMTP邮件中继”。使用SMTP，可实现相同网络处理进程之间的邮件传输，也可通过中继器或网关实现某处理进程与其他网络之间的邮件传输。

python的smtplib提供了一种很方便的途径发送电子邮件。它对smtp协议进行了简单的封装。

安装

python3自带，无需安装

引入

import smtplib
from email.mime.text import MIMEText
from email.header import Header

简单发送邮件实例

目标：发送1封邮件，无附件

定义发送内容、接受与发送地址

首先定义内容，分别为：

  • 邮件服务器地址：smtp.xxx.com，xxx的内容填邮箱的号，例如：163、139等
  • 发送方邮箱
  • 发送方密码，强烈推荐授权密码，这个去相应的邮箱地址获取即可
  • 收件方邮箱
  • 邮件标题
  • 邮件内容
server = 'smtp.163.com' # 发送邮件服务器地址
sender = 'xxxx@163.com' # 发送方账号
sender_password = 'xxxxxx' # 发送方密码（或授权密码）
receiver = 'xxxxx@163.com' # 收件方邮箱
title = 'Python SMTP 测试邮件' # 邮件标题
content = 'Python 测试邮件发送。。。。' # 邮件内容

使用MIMEText(_text,_subtype,_charset)方法，三个参数的作用如下：

  • _text：文本内容
  • _subtype：设置文本格式，默认为plain，若为html则设置文本格式为html格式
  • _charset：设置编码，通常为utf-8
message = MIMEText(content, 'plain', 'utf-8')  # 发送内容 （文本内容，发送格式，编码格式）
message['From'] = sender # 发送地址
message['To'] = receiver # 接受地址
message['Subject'] = Header(title,'utf-8') # 邮件标题

开始发送

使用SMTP进行发送，步骤为：

  1. 创建SMTP对象
  2. 连接服务器
  3. 登录邮箱账号
  4. 发送邮件
  5. 关闭smtp
try:
    smtp = smtplib.SMTP() # 创建SMTP对象
    smtp.connect(server) # 连接服务器
    smtp.login(sender, sender_password) # 登录邮箱账号
    smtp.sendmail(sender, receiver, message.as_string()) # 发送账号信息
    print('发送成功')
except smtplib.SMTPException:
    print('邮件发送失败')
finally:
    smtp.quit()

发送多封带有附件的邮件实例

目标：发送多封邮件，有附件，目标为不同的多个人，邮件格式为html

引入MIMEMultipart

from email.mime.multipart import MIMEMultipart

定义发送内容、接受与发送地址

首先定义内容，分别为：

  • 邮件服务器地址：smtp.xxx.com，xxx的内容填邮箱的号，例如：163、139等
  • 发送方邮箱
  • 发送方密码，强烈推荐授权密码，这个去相应的邮箱地址获取即可
  • 收件方邮箱
  • 邮件标题
  • 邮件内容，注意用html书写
server = 'smtp.163.com' # 发送邮件服务器地址
sender = 'xxxx@163.com' # 发送方账号
sender_password = 'xxxxxx' # 发送方密码（或授权密码）
receiver = ['xxxxx@163.com','xxxxx@163.com'] # 收件方邮箱，用列表存储
title = 'Python SMTP 测试邮件' # 邮件标题
message = "<p>Python 邮件发送测试...</p> <p><a href=\"http://www.baidu.com\">这是一个链接</a></p>" # 邮件内容

使用MIMEMultipart()方法，首先添加正文：

message = MIMEMultipart()  # 创建一个带附件的实例
message['From'] = sender # 发送地址
message['To'] = ','.join(receiver) # 接受地址
message['Subject'] = Header(title,'utf-8') # 邮件标题
message.attach(MIMEText(content, 'html', 'utf-8')) # 添加邮件正文内容

其次添加附件：

appendix = MIMEText(open('12138.txt', 'rb').read(), 'base64', 'utf-8') # 构造附件，传送当前目录下的12138.txt 文件
appendix["Content-Type"] = 'application/octet-stream' # 此处为固定的格式，可以在浏览器中查看到相关信息
appendix["Content-Disposition"] = 'attachment; filename="12138.txt"' # 这里的 filename 命名任意，即在邮件中显示的名称
message.attach(appendix) # 添加附件

当然也可以增加另外一个附件，再次使用上面的方法即可。

  1. 创建SMTP对象

  5. 关闭smtp

    print('发送成功')
except smtplib.SMTPException:
    print('邮件发送失败')
finally:
    smtp.quit()

工具类整合

直接复制即可

# -*- encoding=utf8 -*-  
import smtplib  
from email.mime.image import MIMEImage  
from email.mime.multipart import MIMEMultipart  
from email.mime.text import MIMEText  
from email.header import Header  
import os  
  
  
class EmailSenderUitls:  
    """邮件工具类"""  
  
    def spilt_path_filename(self,bar_type,filepath):  
        """  
        通过文件路径切割文件路径与文件名称  
        在调用该类方法时，建议用它切割文件路径  
  
        :param bar_type: 斜杆的类型，与列表中所有的文件路径对应。eg.D:/bar.txt，那么该参数为'/'；D:\bar.txt，那么该参数为'\'  
        :param filepath: 文件路径，!!!必须是列表!!!，斜杆类型必须一致，eg.["D:/bar.txt","D:/bar/bar.txt"]  
        :return: 是否发送成功，成功为1，不成功为0  
        """        filepath_list = []  
        filename_list = []  
        for path in filepath:  
            split_path = os.path.split(path)  
            file_path = split_path[0] + bar_type  
            file_name = split_path[1]  
            filepath_list.append(file_path)  
            filename_list.append(file_name)  
        return [filepath_list,filename_list]  
  
    def sendEmail_Text(self,sender,sender_password,receiver,title,content,smtp_server):  
        """  
        发送纯文本邮件  
  
        :param sender: 发送人邮件  
        :param sender_password: 发送人邮件的密码或授权码  
        :param receiver: 接收人邮件，可以为多个，用列表填充  
        :param title: 邮件标题  
        :param content: 邮件正文内容  
        :param smtp_server: 邮件发送的服务器  
        :return: 是否发送成功，成功为1，不成功为0  
        """        message = MIMEText(content, 'plain', 'utf-8')  # 发送内容 （文本内容，发送格式，编码格式）  
        message['From'] = sender  # 发送地址  
        message['To'] = ','.join(receiver)  # 接受地址  
        message['Subject'] = Header(title, 'utf-8')  # 邮件标题  
  
        try:  
            smtp = smtplib.SMTP()  # 创建SMTP对象  
            smtp.connect(smtp_server)  # 连接服务器  
            smtp.login(sender, sender_password)  # 登录邮箱账号  
            smtp.sendmail(sender, receiver, message.as_string())  # 发送账号信息  
            return True  
        except smtplib.SMTPException as e:  
            print(e)  
            return False  
        finally:  
            smtp.quit()  
  
  
    def sendEmail_Html(self,sender,sender_password,receiver,title,content_html,smtp_server):  
        """  
        发送HTML格式邮件，强烈推荐用这个发送纯文本邮件  
  
        :param sender: 发送人邮件  
        :param sender_password: 发送人邮件的密码或授权码  
        :param receiver: 接收人邮件，可以为多个，用列表填充  
        :param title: 邮件标题  
        :param content_html: 邮件正文内容，HTML格式，这会在邮件上显示格式  
        :param smtp_server: 邮件发送的服务器  
        :return: 是否发送成功，成功为1，不成功为0  
        """        message = MIMEText(content_html, 'html', 'utf-8')  # 发送内容 （文本内容，发送格式，编码格式）  
        message['From'] = sender  # 发送地址  
        message['To'] = ','.join(receiver)  # 接受地址  
        message['Subject'] = Header(title, 'utf-8')  # 邮件标题  
  
        try:  
            smtp = smtplib.SMTP()  # 创建SMTP对象  
            smtp.connect(smtp_server)  # 连接服务器  
            smtp.login(sender, sender_password)  # 登录邮箱账号  
            smtp.sendmail(sender, receiver, message.as_string())  # 发送账号信息  
            return True  
        except smtplib.SMTPException as e:  
            print(e)  
            return False  
        finally:  
            smtp.quit()  
  
    def sendEmail_file(self,sender,sender_password,receiver,title,content_html,file_path,file_name,smtp_server):  
        """  
        发送HTML格式邮件，并带上附件  
  
        :param sender: 发送人邮件  
        :param sender_password: 发送人邮件的密码或授权码  
        :param receiver: 接收人邮件，可以为多个，用列表填充  
        :param title: 邮件标题  
        :param content_html: 邮件正文内容，HTML格式，这会在邮件上显示格式  
        :param file_path: 附件路径，只是路径不包括附件名，如src/main/，!!!必须用列表存!!!，eg.['src/main/','src/jar/']  
        :param file_name: 附件名称，!!!必须用列表存!!!，每个附件名称的下标与路径列表对应，eg.['1.txt','2.jar']，与附件路径例子组合成src/main/1.txt；src/jar/2.jar  
        :param smtp_server: 邮件发送的服务器  
        :return: 是否发送成功，成功为1，不成功为0  
        """        message = MIMEMultipart()  # 创建一个带附件的实例  
        message['From'] = sender  # 发送地址  
        message['To'] = ','.join(receiver)  # 接受地址  
        message['Subject'] = Header(title, 'utf-8')  # 邮件标题  
        message.attach(MIMEText(content_html, 'html', 'utf-8'))  # 添加邮件正文内容  
  
        i = 0  
        while(i < len(file_path)):  
            with open(file_path[i]+file_name[i], 'rb') as file:  
                appendix = MIMEText(file.read(), 'base64', 'utf-8')  # 构造附件，传送文件  
                appendix["Content-Type"] = 'application/octet-stream'  # 此处为固定的格式，可以在浏览器中查看到相关信息  
                appendix["Content-Disposition"] = f'attachment; filename="{file_name[i]}"'  # 这里的 filename 命名任意，即在邮件中显示的名称  
                message.attach(appendix)  # 添加附件  
                file.close()  
            i = i+1  
  
        try:  
            smtp = smtplib.SMTP()  # 创建SMTP对象  
            smtp.connect(smtp_server)  # 连接服务器  
            smtp.login(sender, sender_password)  # 登录邮箱账号  
            smtp.sendmail(sender, receiver, message.as_string())  # 发送账号信息  
            return True  
        except smtplib.SMTPException as e:  
            print(e)  
            return False  
        finally:  
            smtp.quit()

        message['From'] = sender  # 发送地址
